## How to Capture

1. Open the generated diagram in Modelio
2. Use **File > Export as Image** or take a screenshot
3. Save as PNG with a descriptive name
4. Place in this folder

## Image Guidelines

- Use PNG format for best quality
- Crop to show just the diagram (no Modelio UI)
- Width: 800-1200 pixels recommended
- Include the full process from start to end
